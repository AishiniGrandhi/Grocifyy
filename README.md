# Grocifyy
Grocery Delivery Website

## Description
Grocifyy is a simple grocery delivery website developed as a college project.
The website demonstrates basic front-end concepts using HTML, CSS, and JavaScript.

## Features
- Product category pages (Fruits, Vegetables, Dairy, Meat)
- Cart and checkout pages
- Order tracking page
- Simple and responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in a web browser

## Academic Use
This project was created for learning and academic purposes.
